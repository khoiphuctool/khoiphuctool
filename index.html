📖 Hướng dẫn sử dụng

1

Tải xuống

Nhấn nút tải xuống, lưu file .exe về máy. Không cần cài đặt.

2

Đăng nhập

Mở app, nhập mật khẩu tài khoản FREE hoặc PRO để sử dụng.

3

Chọn file

Chọn file cần khôi phục mật khẩu (Excel, Word, RAR, ZIP...).

4

Cấu hình

Chọn bộ ký tự, độ dài hoặc file từ điển phù hợp.

5

Bắt đầu

Nhấn Start và chờ kết quả. Mật khẩu tìm được hiển thị ngay.

⚠️ Lưu ý: Các công cụ này chỉ dành cho mục đích khôi phục mật khẩu của chính bạn. Không sử dụng để truy cập trái phép vào tài khoản hoặc file của người khác. Tác giả không chịu trách nhiệm về việc sử dụng sai mục đích.
